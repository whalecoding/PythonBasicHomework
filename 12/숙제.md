- 아래 코드의 실행 결과는 무엇인지 적어보세요.

```python
class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age
  
  def get_name(self):
    return self.name
    
  def get_age(self):
    return self.age
    
a = Person('철수', 13);
b = Person('영희', 12);
 
print('{}는 {}살 입니다.'.format(a.get_name(), a.get_age()))
print('{}는 {}살 입니다.'.format(b.get_name(), b.get_age()))
```

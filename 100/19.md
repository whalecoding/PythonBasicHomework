### 개념
```python
for 변수 in range(초기값, ~하는 동안, 순환할 때마다 증가하는 값):
    수행할 문장1
    수행할 문장2
    ...
```

range()의 기본값은 다음과 같습니다.       
```python
range(0, ..., 1)
```

|예시|의미|범위|
|------|---|---|
|range(3, 7, 1)|range(3, 7, 1)  |3, 4, 5, 6|
|range(3, 7)|range(3, 7, 1)  |3, 4, 5, 6|
|range(7)|range(0, 7, 1)  |0, 1, 2, 3, 4, 5, 6|
